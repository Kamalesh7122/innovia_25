# INNOVIA'25 - AIML Department Event Website

Frontend-only website developed for the AIML Department's technical and non-technical events, including Hackathon, TriTech Quest, and Speak & Seek. Integrated with Google Forms to collect participant registration details efficiently and provide a smooth event experience.

## Features

- Responsive Design - Fully optimized for desktop, tablet, and mobile devices
- Dynamic Countdown Timer - Real-time countdown to event start
- Smooth Scroll Effects - Parallax background and smooth navigation
- Interactive UI - Animated buttons, hover effects, and scroll progress bars
- Event Pages - Dedicated pages for each event with detailed information

## Tech Stack

- HTML5 - Semantic markup
- CSS3 - Custom properties, flexbox, grid, animations
- JavaScript (ES6) - Vanilla JS for interactivity
- Font Awesome - Icons
- Google Fonts - Orbitron & Montserrat typography

## Installation

1. Clone the repository

git clone https://github.com/Kamalesh7122/innovia_25.git
cd innovia_25


2. Open `index.html` in your browser
